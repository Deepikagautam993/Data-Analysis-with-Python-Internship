# Student Performance Data Analysis - Task 1

## 📌 Project Overview
This project is part of my **Data Science Internship at Maincrafts Technology**. The goal of this task is to perform an Exploratory Data Analysis (EDA) on the "Student Performance Dataset" to identify key factors influencing students' final academic grades (G3).

## 📊 Dataset Description
The dataset contains student demographic, social, and school-related features. 
- **Attributes:** School, gender, age, family size, study time, failures, absences, etc.
- **Target Variable:** `G3` (Final Grade).

## 🛠️ Technologies Used
- **Language:** Python
- **Environment:** Google Colab
- **Libraries:** - `Pandas`: Data manipulation and cleaning.
  - `Matplotlib` & `Seaborn`: Data visualization.
  - `NumPy`: Numerical operations.

## 🔎 Key Insights from Analysis
During the analysis, I discovered the following:
1. **Academic Correlation:** There is a very strong positive correlation between `G1` (First period grade), `G2` (Second period grade), and the final grade `G3`.
2. **Study Time:** Students who dedicated more hours to studying generally achieved higher grades.
3. **Absences:** A negative correlation was observed between the number of school absences and final academic performance.
4. **Gender Performance:** Visualization showed slight variations in grading distributions between Male and Female students.

## 📈 Visualizations Included
- **Distribution Plots:** To understand the spread of final grades.
- **Box Plots:** To compare performance across categories like Gender.
- **Bar Plots:** To analyze the impact of Study Time.
- **Correlation Heatmap:** To see the relationship between all numeric variables at once.

## 🚀 How to Run the Project
1. Clone this repository or download the `.ipynb` file.
2. Upload the file to [Google Colab](https://colab.research.google.com/).
3. Upload the `student-mat.csv` dataset.
4. Run all cells to see the analysis and visualizations.
