# Data Analysis with Python Internship – Tasks Repository

This repository contains the tasks completed as part of my Data Science Internship at Maincrafts Technology.

# Task 1: Student Performance Data Analysis

## Project Overview

This task focuses on performing Exploratory Data Analysis (EDA) on the Student Performance Dataset to identify key factors influencing students' final academic grades (G3).

## Dataset

Student Performance Dataset containing student demographic and academic information.

## Key Steps Performed

* Data Loading
* Data Cleaning
* Handling Missing Values
* Data Analysis
* Data Visualization

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* NumPy
